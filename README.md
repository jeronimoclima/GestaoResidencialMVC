

 Dashboard de Gestão Financeira - ASP.NET Core MVC
Este projeto é uma aplicação de gerenciamento financeiro desenvolvida para facilitar o controle de receitas e despesas. A aplicação oferece uma visão clara da saúde financeira através de um dashboard intuitivo e uma gestão detalhada de transações.

 Funcionalidades
Resumo Geral: Cards dinâmicos que exibem o Total de Receitas, Total de Despesas e o Saldo Atual.

Gestão de Lançamentos: CRUD completo para registrar entradas e saídas.

Interface Responsiva: Tabela interativa com badges coloridos para identificação rápida de tipos de transação.

Segurança de Dados: Validação de formulários via ModelState.

 Tecnologias e Ferramentas
Backend: C# com .NET (ASP.NET Core MVC).

Banco de Dados: Entity Framework Core com SQL Server (Migrations para versionamento).

Frontend: HTML5, Razor Pages e CSS3.

Estilização: Bootstrap 5 para o layout e Grid System.

Ícones: Bootstrap Icons.
